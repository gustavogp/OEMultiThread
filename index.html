Welcome to Order Entry GitHub Page.

Oemultithead is a Java app consisting of the following features:

  • Content extraction from different file formats (PDF, doc, docx, xlx, xlsx, rtf, html, txt)
  • Identification of Part Number, quantity and Price using an Evolutionary Algorithm.
  • Create a temporary xml output file.
  • Create an Excel template from the xml, as well as csv files for flat file EDI via Sterling.
  • Generate individual messages to send order files to arquive.
